# Management Artifacts

This directory will hold planning and status conventions for projects using `agentic-dev-os`.

Management artifacts should answer:

- What is the approved master plan?
- What tasks exist?
- Which task is ready for execution?
- What is blocked?
- What has been reviewed?
- What has been validated?

For the initial templates, use `templates/project/MASTER_PLAN.md`, `templates/project/TASK_BOARD.md`, and `templates/project/VALIDATION_STATUS.md`.
