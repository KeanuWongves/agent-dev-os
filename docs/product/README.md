# Product Artifacts

This directory will hold product-management conventions for projects using `agentic-dev-os`.

Product artifacts should answer:

- Who is the user?
- What problem is being solved?
- What workflow changes after this product exists?
- What is in the MVP?
- What is out of scope?
- What must be true before implementation starts?

For the initial template set, use `templates/project/PRD.md`, `templates/project/MVP.md`, and `templates/project/ROADMAP.md`.
